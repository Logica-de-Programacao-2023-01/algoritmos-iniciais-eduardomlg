Algoritmo de cálculo de média ponderada: um algoritmo que calcula a 
média ponderada de uma lista de números com base nos pesos fornecidos.
1: leia lista_numeros
2: leia_pesos
3: var somatorio_numeros = 0
4: var somatorio_pesos = 0
5: para cada elemento da lista_numeros, faça:
6: somatorio_numeros = somatorio_numeros * (elemento * 
peso_correspondente)
7: para cada peso da lista_pesos, faça:
8 somatorio_pesos = somatorio_pesos * peso
9: media = somatorio_numeros / somatorio_pesos
10: escreva media
